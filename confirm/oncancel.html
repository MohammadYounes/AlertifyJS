Confirm Dialog

A confirm dialog is often used if you want the user to verify or accept something. When a confirm dialog pops up, the user will have to click either "OK" or "Cancel" to proceed.

oncancel { Function }

Description: Gets or sets a callback function to be invoked when the user clicks 'Cancel' button or closes the dialog.

Default: undefined

 alertify.confirm('a callback will be invoked on cancel.').set('oncancel', function(closeEvent){ alertify.error('Cancel');} );
Run

Remarks

To prevent the dialog from closing, either set closeEvent.cancel = true or make your callback return false .
